MATA_ 커미션을 이용해주셔서 감사합니다! 
커미션을 신청하시기 전에 아래의 내용을 먼저 읽어주시기 바랍니다.
커미션 진행 시 해당 내용 숙지 및 환불 정책에 동의 한 것으로 간주되며, 해당 내용 미숙지로 인해 발생하는 문제는 책임지지 않습니다.

*커미션 작업 내역은 모두 해당 리포지토리에 저장되며, 공개됩니다.



== 신청 관련 ==
1. 신청하고자 하는 캐릭터의 이미지를 문의 채팅방에 업로드 하셔야 하며, 여러 구도가 포함된 이미지를 업로드 하시면 더욱 원활한 작업이 가능합니다.

2. 배경은 단순한 배경만 가능합니다. 따로 배경 신청이 없다면 캐릭터에 어울리는 색상의 단색 배경이 제공됩니다.

3. 유혈, 료나, 스캇, 메카닉, 수인, 만화 형식, R-18은 받지 않습니다.

4. 원하시는 포즈 및 구도를 신청하실 경우, 예시 이미지를 함께 업로드 하시면 더욱 원활한 작업이 가능합니다. 단, 과도하게 복잡할 경우 신청 불가할 수 있습니다.
   


== 작업 관련 ==
1. 모든 작업은 선 입금 후 진행됩니다. 입금이 확인되지 않으면 작업이 진행되지 않습니다.

2. 작업 순서는 다음과 같습니다.
   신청 -> 러프(밑 그림) -> 확인 -> 선화 -> 확인 -> 채색 -> 확인 -> 최종 수정 및 완성

3. 컨펌(수정)은 러프 단계에서 무제한으로 가능합니다. 단, 캐릭터 변경, 구도 변경, 포즈 변경등의 대규모 수정은 불가합니다.
   선화 단계부터 최대 3회로 제한되며, 3회 초과 시 더 이상 컨펌이 불가합니다.
 
2. 기본 작업 기간은 7일 입니다. 이 기간은 변경되지 않으며, 초과 시 환불해 드립니다.

3. 커미션 작업 이미지에 대한 저작권은 신청자에게 귀속됩니다. 단, 상업 캐릭터(예: 게임 캐릭터, 애니 캐릭터 등)의 경우 저작권이 신청자에게 귀속될 수 없습니다.

4. 모든 커미션 작업물은 커미션 채널을 포함한 그림 관련 채널과 MATA_ 픽시브에 업로드 됩니다.

5. 작업 이미지 사이즈는 4000px x 3000px, 600DPI 입니다. 신청 내용에 따라 이 사이즈는 변경될 수 있습니다.

6. 사용하는 프로그램은 PaintTool SAI 2입니다.



== 환불 정책 ==
1. 전액 환불은 작업 기간(7일)이 초과되거나, 작업 순서가 러프 단계일 때 가능합니다. 
   선화 단계에서는 50%만 환불 가능하며, 채색 단계 이후에는 환불이 불가합니다.

2. 환불은 신청자의 계좌로만 가능하며, 환불 계좌의 명의가 신청자 명의와 일치하지 않을 경우 환불 진행이 불가합니다.

3. 환불 절차 완료 이후 진행 중인 작업물은 삭제되니 이 점 유의하시기 바랍니다.
